# Lark 自动打卡助手

一个基于 PyQt5 的飞书自动打卡工具，支持图像识别、定时任务、配置导入导出等功能。

## 功能特性

- 📷 **图片配置管理**：上传并保存打卡流程所需的截图
- ⏰ **定时任务**：设置上班/下班时间，自动执行打卡
- 🎲 **时间随机化**：支持时间偏差设置，避免固定时间打卡
- 📦 **配置导入导出**：一键导出配置，方便在多台电脑使用
- ⚙️ **高级设置**：自定义识别精度、重试次数、延迟时间等
- 📋 **实时日志**：彩色日志显示，实时查看执行状态

## 使用方法

### 1. 启动应用

```bash
python main.py
```

### 2. 配置图片

在 **图片配置** 标签页：
1. 点击"上传图片"选择截图
2. 点击对应的"保存"按钮，将图片保存为：
   - `lark` - 飞书图标
   - `工作台` - 工作台按钮
   - `假勤` - 假勤应用图标
   - `上班` - 上班打卡按钮
   - `下班` - 下班打卡按钮

### 3. 高级设置（可选）

在 **高级设置** 标签页可以调整：
- 图像识别相似度阈值（默认 0.8）
- 识别失败重试次数（默认 3 次）
- 步骤间延迟时间（默认 2 秒）
- 启动后最小化窗口
- 任务失败时系统通知

### 4. 设置时间

在主界面右侧：
- 设置上班时间（如 09:00）
- 设置下班时间（如 18:00）
- 设置时间偏差（如 ±5 分钟）

### 5. 启动任务

- 点击 **"启动任务"** 开始自动打卡
- 点击 **"测试上班"** 或 **"测试下班"** 立即测试流程

### 6. 导出/导入配置

- **导出配置**：将所有图片打包为 `.zip` 文件
- **导入配置**：在其他电脑上导入配置，无需重新截图

## 工作原理

1. **图像识别**：使用 OpenCV 模板匹配技术在屏幕上查找目标图片
2. **自动点击**：使用 PyAutoGUI 模拟鼠标点击
3. **定时任务**：使用 QTimer 定时检查是否到达执行时间
4. **多线程**：UI 线程和工作线程分离，避免界面卡顿

## 执行流程

```
启动任务 → 等待执行时间 → 寻找 Lark 图标 → 点击打开
         ↓
    寻找工作台 → 点击进入
         ↓
    寻找假勤 → 点击进入
         ↓
    寻找上班/下班按钮 → 点击打卡 → 完成
```

## 注意事项

1. **屏幕分辨率**：截图时的分辨率应与实际使用时一致
2. **窗口位置**：飞书窗口位置变化可能导致识别失败
3. **图片质量**：截图应清晰，避免模糊或遮挡
4. **权限设置**：某些系统可能需要授予屏幕录制权限
